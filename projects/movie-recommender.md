## Overview

A recommendation engine that suggests similar movies from a user's input, using
**TF-IDF vectorization** and **cosine similarity** over film plot descriptions, with an
interactive **Streamlit** front end.

> Replace this with your Notion write-up — `projects/movie-recommender.md`.

## How it works

- Vectorizes movie plot descriptions with **TF-IDF**
- Measures similarity between films using **cosine similarity**
- Returns the closest matches for any selected movie
- Search and results presented through a Streamlit UI
