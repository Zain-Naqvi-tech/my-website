About

I'm a Computer Engineering student at McMaster University working at the seam of hardware and software. My focus is firmware and embedded systems — and I've shipped real hardware across two robotics teams: McMaster RoboSub and the Aerial Robotics & Drones Club, designing PCBs in Altium and working with protocols like CAN, SPI, I²C, and UART.

Beyond firmware, I like building things that reach all the way up the stack — computer vision, applied ML, and full applications. I'm looking for opportunities in firmware, embedded systems, robotics, and software engineering.

  • Firmware — C / C++ / RTOS
  • Protocols — CAN / SPI / I²C / UART
  • Hardware — PCB Design / Oscilloscopes / MCUs
  • Software — Python / Git / Linux
  • Robotics & AI — ROS / OpenCV / ML
